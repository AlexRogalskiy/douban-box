# douban-box
![Update Gist](https://github.com/CodeDaraW/douban-box/workflows/Update%20Gist/badge.svg)

## 安装
``` sh
npm i -g douban-box
```

## 使用
提供三个环境变量：
| 变量 | 含义 |
|---|---|
| GIST_ID | Gist ID |
| GITHUB_TOKEN | GitHub Token |
| DOUBAN_ID | 豆瓣用户 ID |

## FAQ
### 豆瓣用户 ID 获取方式
个人主页中 people 后紧接的那串数字或者自定义字符，例如下面链接中是 `daraw`：
```
https://www.douban.com/people/daraw/
```
